# 武雄市におけるこれまでの施策をみてみましょう {.lead}

## 官民一体型学校（はなまる学習会との提携）

<ul class="lead">
<li>3期目当選後に、官民一体型学校（はなまる学習会との提携）の構想を発表</li>
<li><strong>市長選挙期間中を含む発表以前に、上記施策についての市民への説明は、なし</strong></li>
<li>公募によらず、はなまる学習会と10年間の契約を提携<br />
  <small>（武雄市がはなまる学習会から教材を購入）</small></li>
</ul>

-----

市長選があったにも関わらず、教育に関わる重要施策を秘密にして選挙に臨み、当選直後に発表するという手法をとっています。
また、通常自治体おいては金額が大きくなる場合、公募によって契約業者を決めますがそれを行っていません。

## 民間人校長の登用

<ul class="lead">
<li>佐賀県内初の民間人校長を登用</li>
<li>任命された人物は杉並区立和田中学校の元校長</li>
<li>同人物を教育長に任命しようとした鎌倉市では任命を巡り様々な問題や議論が引き起こされた</li>
</ul>
